Enter the site that contains the list to work on in the "site" column.
Specify the list to set the default view on in the "list" column.
Specify the name of the new view in the "view_name" column.
Set the field to sort on in the "sort_field" column.
Set the column to index on, if any, in the "index_on" column. Leave blank to not index on a column.
Set the direction to sort on in the "direction" column.
Set the limit for displayed items in the "limit" column.
